# Temperature sensor
This repository is about to develop a mobile temperature sensor using an ESP8266 microcontroller with a temp. and humidity sensor DHT22 etc. <br>
Additionally there is the plan to use a microSD add on

## Hardware setup
* D1 mini ESP8266 WiFi Board (e.g. ebay -> makershop or makershop.de) - price: 12,75 € (3pcs) <br>
* DHT22 temperature and humidity sensor shield for WeMos D1 mini (e.g. ebay -> makershop or makershop.de) - price: 8,75 € <br>
* DS1307 data logger shield for WeMos D1 mini (e.g. ebay -> makershop or makershop.de) - price: 5,75 € <br>
